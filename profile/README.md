# SHINE Laboratory

## University of Colorado Boulder

The **System-Human Interaction with NIRS and EEG (SHINE) Lab** is an academic research lab centered in the **Institute of Cognitive Science** at the **University of Colorado Boulder**.

Directed by **Dr. Leanne Hirshfield**

[Lab Website](https://www.colorado.edu/lab/shine/) • [Institute of Cognitive Science](https://www.colorado.edu/ics/)

## Code Directory

This GitHub organization hosts SHINE Lab code for former and ongoing projects. Code is organized into repositories based on the type of data being analyzed. Individual repositories include README files with more specific information about the contents and usage of each repository.

Below are the repositories available from the SHINE Lab as well as their contents:

### NIRS_Analysis

Contains code for analysis of fNIRS data. In the SHINE Lab, we primarily utilize **NIRx Sports 2** as well as **Artinis PortaLite**.

### MRI_Analysis

Contains code for analysis of MRI data. The SHINE Lab utilizes a **Siemens Prisma MRI** machine with a strength of **3T**, managed by the [Intermountain Neuroimaging Consortium (INC)](https://www.colorado.edu/mri/).

### EEG_Analysis

Contains code for analysis of EEG data.

### Multimodal Analysis

Contains code for multi-sensor and multimodal analysis.

## Resources and Installation Instructions

Much of our code is written in **MATLAB**, which requires special installation. To install MATLAB using your CU login, head to [this page](https://www.mathworks.com/academia/tah-portal/university-of-colorado-boulder-361635.html) and sign in.
